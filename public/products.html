המוצרים הכיפיים שלנו

גלו את כל מגוון המוצרים שהופכים כל רחצה לחוויה בלתי נשכחת

ברווזי גומי קלאסיים

הברווזים הצהובים האהובים שכולם מכירים! זמינים במגוון גדלים - מברווזים קטנים לתינוקות ועד ברווזים גדולים ומרשימים. כל ברווז עשוי מגומי איכותי, בטיחותי לשימוש ונעים למגע.

  • ✓ עשויים מגומי איכותי ובטיחותי
  • ✓ זמינים בגדלים שונים
  • ✓ צבעים עזים ועמידים
  • ✓ קלים לניקוי ותחזוקה

ערכות רחצה מלאות

ערכות מושלמות לחוויית רחצה שלמה! כל ערכה כוללת מגוון צעצועים: ברווזים, כלי שיט, דמויות ימיות ועוד. מארזים מושלמים למתנה או להתחלת אוסף צעצועי אמבטיה.

  • ✓ מגוון רחב של צעצועים בכל ערכה
  • ✓ אריזה צבעונית ומושכת
  • ✓ מושלם למתנה
  • ✓ ערך מוסף מעולה

צעצועי מים אינטראקטיביים

הוסיפו אקשן לאמבטיה! משאבות מים, מזרקות צבעוניות, גלגלי מים מסתובבים וצעצועים מתיזים שיוצרים חוויית משחק מרגשת ואינטראקטיבית. מפתחים מיומנויות מוטוריות ותיאום עין-יד.

  • ✓ פעילות אינטראקטיבית מהנה
  • ✓ מפתח מיומנויות מוטוריות
  • ✓ בטיחותי לשימוש במים
  • ✓ מעודד משחק יצירתי

דמויות ימיות וחיות

הכירו את חברי הים והיער! מגוון רחב של דמויות חיות מקסימות - דולפינים, צבים, דגים צבעוניים, צפרדעים ועוד. כל דמות בעיצוב ייחודי וצבעוני שמעורר דמיון ומעודד משחק יצירתי.

  • ✓ מגוון רחב של דמויות
  • ✓ עיצובים מקוריים ומקסימים
  • ✓ מעודד משחק דמיוני
  • ✓ מתאים לגילאים שונים
